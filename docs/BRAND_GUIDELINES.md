# ShinGiTai Language - Brand Guidelines

Status: Batch 7.1 Identity System
Owner: ArticSakuraTech / ShinGiTai Holding Groupe
Last updated: 2026-07-03

## Purpose

This document defines the brand foundation for ShinGiTai Language.

ShinGiTai Language is part of the ShinGiTai Holding Groupe ecosystem and must feel connected to future products such as ShinGiTai Hub, ShinGiTai Forge and ShinGiTai OpenAI.

## Brand Position

ShinGiTai Language is an AI-first language learning product.

The product should feel:

- premium,
- focused,
- intelligent,
- calm,
- modern,
- trustworthy,
- useful from the first session.

It should not feel like a generic course website or a temporary prototype.

## Brand Architecture

Current hierarchy:

```text
ShinGiTai Holding Groupe
└── ArticSakuraTech
    └── ShinGiTai Language
```

Future ecosystem alignment:

```text
ShinGiTai Hub
├── ShinGiTai Language
├── ShinGiTai Forge
├── ShinGiTai OpenAI
└── future products
```

## Logo Rule

The logo is a canonical brand asset.

Do not replace it with placeholder icons, random symbols, AI-generated marks, emoji, or temporary approximations.

If the correct approved logo asset is not available in the repository yet, keep the existing visual mark in code and document the missing asset instead of inventing a new one.

Allowed logo states:

- approved ShinGiTai Holding Groupe logo,
- approved ArticSakuraTech logo,
- approved ShinGiTai Language logo,
- approved simplified app icon derived from the logo system.

Disallowed logo states:

- generic sparkle icon as final logo,
- unrelated kanji/katakana symbols,
- random dragon/samurai/AI marks,
- generated icons without founder approval,
- third-party copied logo styles.

## Visual Direction

Default theme: dark.

Visual tone:

- deep dark background,
- electric blue and cyan accents,
- deep violet secondary accent,
- soft gradients,
- subtle glass surfaces,
- readable typography,
- minimal animation,
- no clutter.

Reference feeling, not copying:

- Apple-level polish,
- Linear-level clarity,
- Arc-level modernity,
- Notion-level calm structure,
- VS Code-level developer trust,
- ShinGiTai's own identity on top.

## Color Direction

Canonical product colors are defined in `docs/DESIGN_SYSTEM.md`.

Brand-level intent:

- Electric Blue: AI, clarity, trust.
- Cyan: speech, learning flow, interaction.
- Deep Violet: intelligence, premium technology.
- White: precision and readability.
- Near Black: focus and depth.

## Voice and Copy

Tone:

- direct,
- confident,
- encouraging,
- calm,
- premium,
- not childish.

Avoid:

- overpromising,
- gamified noise everywhere,
- cheap marketing slogans,
- fake urgency.

Good example:

> Continue your language learning journey.

Better future direction:

> Train with your AI language mentor.

## Product Promise

ShinGiTai Language should help users learn through:

- structured lessons,
- AI conversation practice,
- flashcards,
- grammar support,
- personalized study flow,
- future speech and voice training.

The user should feel guided, not overwhelmed.

## Relationship With ShinGiTai Hub

Every major UI and branding decision must be compatible with future Hub integration.

Language should work as:

- standalone app,
- module inside ShinGiTai Hub,
- future desktop app,
- future mobile app.
